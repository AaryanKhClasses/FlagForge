# Sonic Visualiser Tool
---

`Sonic Visualiser` is an application for viewing and analyzing the contents of music and audio sound files. It is designed for musical analysis and can display a wide range of audio features.

It is commonly used by musicians, researchers, and audio engineers for detailed inspection of audio waveforms, spectrograms, and other audio data.

---

## Information
`Sonic Visualiser` provides various visualization panes, including waveform, spectrogram, and pitch views. It allows users to add annotation layers, apply analysis plugins, and export data. This makes it a powerful tool for tasks such as identifying hidden messages in audio, analyzing musical structure, or detecting anomalies in sound.

---

## Using as FlagForge Tool
<p class="text-primary">NOTE: Windows Subsystem for Linux (WSL) might be used for some underlying audio processing utilities, but 'Sonic Visualiser' itself is typically a desktop application. It must be installed separately. The tool can be quickly installed by following instructions given within the app itself.</p>

The `Sonic Visualiser` tool is a FlagForge attachment tool for audio files. Simply select an audio file (e.g., `.wav`, `.mp3`) in the Attachment View and click on the "Attachment Info" button in the top right corner of the main area. Then, click on the "Sonic Visualiser" tool to open the file within the application.

---

## Example Usage
Suppose you have an audio file named `mystery.wav` attached to a challenge and you suspect it contains hidden information in its spectrogram. To analyze it using the `Sonic Visualiser` tool, you would select the file in the Attachment View and click on the "Sonic Visualiser" tool. The application will open, displaying the audio waveform and allowing you to explore its spectrogram for visual patterns or hidden data.
