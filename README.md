## Contributing:
Contributions are welcome! Please fork the repository and submit a pull request with your changes.

### Note for Contributors:
- The project uses React Router for frontend and .NET for backend.
- Make sure you have `Node.JS` (v22 or higher) and `.NET SDK` (v10) installed on your machine.
- Follow the coding style and conventions used in the existing codebase:
    - 4-Space Indentation
    - No Trailing Spaces, Commas, or Semicolons
    - Whiteline at the end of files
    - Other conventions as observed in the codebase.

### To Run the Project (DEVELOPMENT):
1. Clone or fork the repository.
2. Navigate to the project directory in your terminal.
3. Run the frontend:
```bash
cd frontend
npm run dev
```
4. Run the host app:
```bash
cd host
dotnet run
```
